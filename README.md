## 热门图书的功能
- 先写服务端，确保数据能正常返回
- 增加API方法，实现调取数据的功能
- 在哪个组件中应用，如果是一个基础组件需要用这些数据，在使用这个组件的父级中调用这个方法，将数据传递给基础组价。
- 写一个基础组件
    - 创建一个.vue文件
    - 在需要使用这个组件的父级中引用这个组件
    - 在组件中注册
    - 以标签的形式引入
